## 🧪 功能測試用例

### TC-001：正方形模式 - iPhone 12 直向

**前置條件**：
- 設備：iPhone 12
- 解析度：390×844
- 方向：直向
- 模式：正方形（有圖片）

**測試步驟**：
1. 打開遊戲
2. 等待卡片加載
3. 檢查卡片大小
4. 檢查列數和行數
5. 檢查邊距和間距
6. 檢查卡片是否超出邊界

**預期結果**：
- [ ] 卡片大小合理（100-150px）
- [ ] 列數為 2-3
- [ ] 行數為 4-6
- [ ] 邊距為 12-15px
- [ ] 間距為 8-12px
- [ ] 卡片不超出邊界
- [ ] 文字不被切割

**實際結果**：
- [ ] 通過 / [ ] 失敗
