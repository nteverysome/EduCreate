## 🚀 核心特性

### 🔥 v132.0 - 雙軸動態縮放系統

**水平縮放（寬度）**
- 如果卡片超出容器：自動縮小以適應
- 如果卡片在容器內：根據利用率調整（1.0 / 0.95 / 0.85）

**垂直縮放（高度）**
- 垂直利用率 > 95%：縮小 10%（0.9x）
- 垂直利用率 85-95%：保持（1.0x）
- 垂直利用率 70-85%：增加 10%（1.1x）
- 垂直利用率 50-70%：增加 20%（1.2x）
- 垂直利用率 < 50%：增加 30%（1.3x）
