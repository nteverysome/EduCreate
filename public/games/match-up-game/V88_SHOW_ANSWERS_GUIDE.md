# Match-up Game v88.0 - Show Answers 直接在遊戲頁面顯示答案

## 🎯 功能說明

v88.0 改進了 "Show answers" 功能，使其直接在遊戲頁面上顯示所有答案結果，而不是彈出模態框。

### 改動內容

**之前（v87.0 及之前）**：
- 點擊 "Show answers" 按鈕 → 彈出模態框
- 模態框中顯示用戶答案和正確答案
- 需要關閉模態框才能返回

**之後（v88.0）**：
- 點擊 "Show answers" 按鈕 → 直接在遊戲頁面上顯示答案
- 清除所有卡片，顯示答案結果（帶著勾勾和叉叉）
- 添加 "Back to Results" 按鈕返回遊戲結束畫面

## 📋 實現細節

### 1. 修改 "Show answers" 按鈕點擊事件

**文件**: `scenes/game.js` (第 6012-6020 行)

```javascript
// Show answers 按鈕
this.createModalButton(modal, 0, buttonY, 'Show answers', () => {
    console.log('🎮 點擊 Show answers 按鈕');
    // 🔥 v88.0: 隱藏模態框，直接在遊戲頁面上顯示答案
    overlay.destroy();
    modal.destroy();
    this.gameCompleteModal = null;
    this.displayAnswersOnGameBoard();
});
```

### 2. 新增 displayAnswersOnGameBoard() 方法

根據佈局模式調用相應的顯示方法：
- 混合佈局：調用 `displayAnswersOnMixedLayout()`
- 分離佈局：調用 `displayAnswersOnSeparatedLayout()`

### 3. 新增 displayAnswersOnMixedLayout() 方法

在混合佈局上顯示答案：
- 英文卡片（左側）：顯示用戶的答案
- 中文卡片（右側）：顯示正確答案
- 中文文字：顯示中文翻譯

### 4. 新增 displayAnswersOnSeparatedLayout() 方法

在分離佈局上顯示答案：
- 左側：用戶的答案
- 右側：正確答案
- 下方：中文翻譯

### 5. 新增 displayAnswerCard() 方法

顯示單個答案卡片：
- 背景顏色：根據是否正確決定（綠色/灰色）
- 標記：勾勾（✓）或叉叉（✗）
- 文字：答案內容

### 6. 新增 createReturnButton() 方法

創建 "Back to Results" 按鈕：
- 位置：頁面底部中央
- 功能：返回遊戲結束畫面
- 樣式：藍色按鈕，懸停效果

## 🔄 使用流程

1. **遊戲完成** → 顯示遊戲結束畫面
2. **點擊 "Show answers"** → 清除模態框
3. **顯示答案結果** → 在遊戲頁面上顯示所有答案
4. **點擊 "Back to Results"** → 返回遊戲結束畫面

## ✅ 驗證步驟

### 步驟 1: 啟動遊戲
```
http://localhost:3000/games/switcher?game=match-up-game&activityId=YOUR_ACTIVITY_ID
```

### 步驟 2: 完成遊戲
1. 配對所有卡片
2. 點擊 "Submit" 按鈕
3. 等待遊戲完成

### 步驟 3: 點擊 "Show answers"
1. 在遊戲結束畫面點擊 "Show answers" 按鈕
2. **預期結果**：
   - ✅ 模態框消失
   - ✅ 遊戲頁面顯示所有答案
   - ✅ 每個答案都有勾勾或叉叉標記
   - ✅ 顯示中文翻譯

### 步驟 4: 返回遊戲結束畫面
1. 點擊 "Back to Results" 按鈕
2. **預期結果**：
   - ✅ 返回遊戲結束畫面
   - ✅ 可以再次點擊 "Show answers"
