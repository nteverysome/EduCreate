# v121.0 - 在分頁按鈕中間添加分頁選擇器

## 🎯 目標

在"上一頁"和"下一頁"按鈕的中間添加一個分頁選擇器（頁碼選擇器），風格與分頁指示器相同，支援用戶直接選擇頁碼。

## 📝 新增功能

### 1. **分頁選擇器組件**
- 位置：在"上一頁"和"下一頁"按鈕的中間
- 風格：與分頁指示器相同（淺灰色背景，深灰色文字）
- 功能：顯示當前頁碼和總頁數（如 1/3）

### 2. **減少按鈕（−）**
- 位置：分頁選擇器左側
- 功能：進入上一頁
- 顏色：灰色（#cccccc），懸停時變深灰色（#999999）

### 3. **增加按鈕（+）**
- 位置：分頁選擇器右側
- 功能：進入下一頁
- 顏色：灰色（#cccccc），懸停時變深灰色（#999999）

## 🔧 實現細節

### 新增方法：`createPageSelector()`

**參數：**
- `x`: 選擇器中心 X 位置
- `y`: 選擇器中心 Y 位置
- `width`: 選擇器寬度（80px）
- `height`: 選擇器高度（50px）

**功能：**
1. 創建選擇器背景（淺灰色矩形）
2. 創建頁碼文字（如 1/3）
3. 創建減少按鈕（−）
4. 創建增加按鈕（+）
5. 綁定按鈕點擊事件

### 按鈕事件

**減少按鈕點擊：**
- 檢查是否在第一頁
- 如果不是，調用 `goToPreviousPage()`
- 銷毀選擇器

**增加按鈕點擊：**
- 檢查是否在最後一頁
- 如果不是，調用 `goToNextPage()`
- 銷毀選擇器
