# v120.0 - 調整分頁按鈕位置至頂部，與分頁指示器水平對齐

## 🎯 目標

調整"上一頁"和"下一頁"按鈕的位置，使其與分頁指示器（如 4/4）水平對齐，提升用戶界面的視覺一致性。

## 📝 修改內容

### 修改的方法

**`showPaginationButtons()` 方法 (第 6158-6231 行)**

#### 修改前
```javascript
const centerY = height / 2;  // 按鈕位置在螢幕中央

// 上一頁按鈕位置
const prevButtonX = width / 2 - buttonWidth / 2 - spacing / 2;
// 下一頁按鈕位置
const nextButtonX = width / 2 + buttonWidth / 2 + spacing / 2;

// 創建按鈕時使用 centerY
this.createPaginationButton(
    prevButtonX,
    centerY,  // ❌ 螢幕中央
    ...
);
```

#### 修改後
```javascript
// 🔥 [v120.0] 按鈕位置改為頂部，與分頁指示器水平對齐
const buttonY = 20;  // 與分頁指示器相同的 Y 位置

// 上一頁按鈕位置（左側）
const prevButtonX = width / 2 - buttonWidth / 2 - spacing / 2;
// 下一頁按鈕位置（右側）
const nextButtonX = width / 2 + buttonWidth / 2 + spacing / 2;

// 創建按鈕時使用 buttonY
this.createPaginationButton(
    prevButtonX,
    buttonY,  // ✅ 頂部，與分頁指示器對齐
    ...
);
```
