## 🔑 核心概念速查

### 三個關鍵步驟
```javascript
// 1. 移除舊父容器
if (card.parentContainer) {
    card.parentContainer.remove(card);
}

// 2. 添加到新容器
targetContainer.add(card);

// 3. 設置本地座標為 (0, 0)
card.setPosition(0, 0);
```

### 座標系統規則
```
worldX = containerWorldX + cardLocalX
worldY = containerWorldY + cardLocalY
```

### 檢查清單
- [ ] 物體在正確的容器中？
- [ ] 物體的本地座標是 (0, 0)？
- [ ] 物體的世界座標等於容器座標？
