## 🛠️ 調試技巧

### 在 Safari 中查看控制台

1. **啟用 Web 檢查器**
   - 設定 → 進階 → Web 檢查器（打開）

2. **連接到 Mac**
   - 在 Mac 上打開 Safari
   - 菜單 → 開發 → [你的 iPad 名稱]
   - 選擇 match-up-game 頁面

3. **查看控制台**
   - 菜單 → 開發 → 顯示 JavaScript 控制台

### 在 Chrome 中查看控制台

1. **連接到 Android 設備**
   - 在 Android 上打開 Chrome
   - 在 Mac/Windows 上打開 Chrome
   - 進入 `chrome://inspect`
   - 選擇設備和頁面

2. **查看控制台**
   - 點擊 "inspect" 打開開發者工具

## 🐛 常見問題

### Q1: 為什麼卡片大小沒有變化？

**A**: 
1. 檢查瀏覽器緩存
   - 在 Safari 中：設定 → 進階 → 網站資料 → 清除所有
   - 在 Chrome 中：設定 → 隱私權 → 清除瀏覽資料

2. 強制刷新頁面
   - 按住刷新按鈕 → 選擇「硬性重新載入」

3. 檢查控制台是否有錯誤
   - 打開開發者工具查看是否有 JavaScript 錯誤

### Q2: 不同 iPad 上的卡片大小不同，這是正常的嗎？

**A**: 是的，這是預期行為。v42.0 的目的就是根據 iPad 尺寸調整卡片大小。

### Q3: 文字大小看起來太大/太小？

**A**: 
1. 檢查控制台輸出的 `baseFontSize` 值
2. 如果需要調整，修改 `getIPadOptimalParams()` 函數中的 `chineseFontSize` 值
3. 提交新的改動
