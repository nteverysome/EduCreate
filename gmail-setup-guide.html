🚀 Gmail SMTP 設定指南

為 EduCreate 郵箱驗證功能設定 Gmail SMTP（完全免費）

📧 步驟 1: 準備 Gmail 帳戶

您需要一個 Gmail 帳戶來發送驗證郵件。

🔒 步驟 2: 啟用兩步驟驗證

Gmail SMTP 需要兩步驟驗證才能使用應用程式密碼。

  1. 在 Google 帳戶頁面，點擊左側的「安全性」
  2. 找到「兩步驟驗證」選項
  3. 如果未啟用，請點擊「開始使用」
  4. 按照指示完成設定（通常需要手機號碼）
⚠️ 如果已經啟用兩步驟驗證，可以直接進行下一步。

🔑 步驟 3: 生成應用程式密碼

為 EduCreate 生成專用的應用程式密碼。

  1. 在兩步驟驗證頁面，找到「應用程式密碼」
  2. 點擊「應用程式密碼」
  3. 選擇應用程式：「郵件」
  4. 選擇裝置：「其他（自訂名稱）」
  5. 輸入名稱：「EduCreate SMTP」
  6. 點擊「產生」
  7. 複製顯示的 16 位數密碼

📝 步驟 4: 更新環境配置

將 Gmail 憑證添加到 EduCreate 配置中。

本地開發環境 (.env 文件):

EMAIL_SERVER_HOST="smtp.gmail.com"
EMAIL_SERVER_PORT="587"
EMAIL_SERVER_USER="your-email@gmail.com"
EMAIL_SERVER_PASSWORD="your-app-password"
EMAIL_SERVER_SECURE="true"
EMAIL_FROM="your-email@gmail.com"

🌐 步驟 5: 生產環境配置

在 Vercel 中設定環境變數。

  1. 前往 Vercel 儀表板
  2. 選擇 EduCreate 專案
  3. 點擊「Settings」→「Environment Variables」
  4. 添加以下環境變數：
EMAIL_SERVER_USER = your-email@gmail.com
EMAIL_SERVER_PASSWORD = your-app-password
EMAIL_FROM = your-email@gmail.com

🧪 步驟 6: 測試郵箱驗證

測試 Gmail SMTP 是否正常工作。

🎉 設定完成！現在您可以：
  • 在 EduCreate 註冊新帳戶
  • 檢查郵箱收到驗證郵件
  • 點擊驗證連結啟用帳戶
