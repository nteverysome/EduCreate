### 1024×768 白屏問題 - ✅ 已解決

**原始問題**:
- 1024×768 分辨率顯示白屏
- 設備檢測邏輯誤判為 iPad
- 卡片寬度計算過大導致重疊

**解決方案**:
```javascript
// 改進的設備檢測邏輯
const isDesktopXGA = width === 1024 && height === 768;
const isRealTablet = width >= 768 && width <= 1024 && height >= 600 && !isDesktopXGA;
const isIPad = isRealTablet;
```
