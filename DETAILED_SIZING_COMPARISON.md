## 🔄 **尺寸計算邏輯**

### 卡片寬度計算
```javascript
// 基於容器寬度
const leftContainerWidth = containerWidth * 0.25;
const cardWidth = leftContainerWidth - (sideMargin * 2);
// 例: 1841 * 0.25 - 30 = 430px（單列）
```

### 卡片高度計算
```javascript
// 基於容器高度和卡片數量
const availableHeight = containerHeight * 0.8;
const cardHeight = (availableHeight - spacing * (itemCount - 1)) / itemCount;
// 例: (674 * 0.8 - 10 * 3) / 4 = 75px
```

### 內容大小計算
```javascript
// 語音按鈕
audioButtonSize = cardHeight * 0.25;

// 圖片
imageWidth = cardWidth * 0.35;
imageHeight = cardHeight * 0.5;

// 文字
fontSize = cardHeight * 0.22;
lineHeight = cardHeight * 0.28;

// 邊距
padding = cardHeight * 0.1;
gap = cardHeight * 0.08;
```
