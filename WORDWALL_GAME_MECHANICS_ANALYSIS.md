# Wordwall 遊戲機制深度分析

## 🎮 Wordwall 遊戲類型全景

### 按記憶科學分類

**基礎記憶類型（Foundational Memory）**
- **Quiz（測驗）** - 選擇題、填空題、判斷題
- **Flashcards（閃卡）** - 翻卡記憶、主動回憶
- **Word search（單字搜索）** - 視覺搜索、空間記憶

**配對記憶類型（Associative Memory）**
- **Match up（配對）** - 拖拽配對、視覺配對
- **Anagram（字謎）** - 字母重組、邏輯推理
- **Unjumble（解亂序）** - 單字排序、序列記憶

**動態反應記憶類型（Dynamic Response Memory）**
- **Whack-a-mole（打地鼠）** - 快速反應、時間壓力
- **Maze chase（迷宮追逐）** - 空間導航、動態反應
- **Spin the wheel（轉輪盤）** - 隨機選擇、快速決策

**其他類型**
- **Open the box（打開盒子）** - 隨機揭示、驚喜機制
- **Wordsearch（單字搜索）** - 視覺搜索、專注力

```

  ├─ 交互元素

  ├─ 接收用戶輸入
  ├─ 驗證答案
  └─ 更新狀態
  ↓
反饋系統
  ├─ 正確反饋（綠色、音效、動畫）

  └─ 分數更新
  ↓

  ├─ 未完成 → 下一題
  └─ 完成 → 結束遊戲
  ↓
結果頁面
  ├─ 最終分數
  ├─ 正確率
  ├─ 排行榜排名
  └─ 分享選項
```

## 📊 評分系統

### 評分計算公式

```
基礎分數 = 100 分

時間獎勵 = 基礎分數 × (剩餘時間 / 總時間)
正確率獎勵 = 基礎分數 × (正確答案數 / 總答案數)
速度獎勵 = 基礎分數 × (平均答題速度係數)

最終分數 = 基礎分數 + 時間獎勵 + 正確率獎勵 + 速度獎勵
```

### 排行榜機制

- **實時排名**：按分數排序
- **多人競爭**：支援同時遊玩
- **班級排行榜**：教師可查看班級排名
- **個人最佳**：記錄個人最高分
- **分享功能**：學生可分享成績

## 🎯 核心遊戲機制

### 1. Match up（配對）
```
機制：拖拽左側項目到右側對應項目
反饋：實時驗證，顏色變化
難度：可調整項目數量
記憶類型：視覺配對、關聯記憶
```

### 2. Quiz（測驗）
```
機制：顯示問題，選擇答案
反饋：立即反饋，進度條
難度：可調整題目數量和難度
記憶類型：主動回憶、選擇記憶
```

### 3. Flashcards（閃卡）
```
機制：顯示正面，點擊翻轉
反饋：視覺翻轉動畫
難度：可調整卡片數量
記憶類型：主動回憶、視覺記憶
```

### 4. Whack-a-mole（打地鼠）
```
機制：隨機出現目標，快速點擊
反饋：計分、計時
難度：可調整速度和數量
記憶類型：快速反應、動態記憶
```

## 📱 用戶體驗流程

### 教師視角
```
1. 創建內容集合（詞彙對）
2. 選擇遊戲模板
3. 配置遊戲選項
   ├─ 時間限制
   ├─ 難度級別
   ├─ 隨機順序
   └─ 排行榜設置
4. 生成遊戲鏈接
5. 分享給學生
6. 實時監控進度
7. 查看排行榜
8. 分析學習數據
```

### 學生視角
```
1. 打開遊戲鏈接
2. 輸入名字（可選）
3. 開始遊戲
4. 完成所有題目
5. 查看最終分數
6. 查看排行榜排名
7. 分享成績（可選）
8. 重新遊玩（可選）
```

## 🧠 記憶科學應用

### 主動回憶原理
- 每個問題都要求用戶主動回憶
- 不是被動閱讀
- 立即反饋強化記憶

### 間隔重複
- 支援重複遊玩
- 每次遊玩都是新的間隔
- 強化長期記憶

### 多感官學習
- **視覺**：圖片、文字、顏色
- **聽覺**：音效、語音
- **動覺**：拖拽、點擊、手勢

### 遊戲化激勵
- 分數系統
- 排行榜競爭
- 成就感
- 社交分享

## 💻 技術架構

### 前端技術棧
- **渲染引擎**：HTML5 Canvas / WebGL
- **遊戲邏輯**：JavaScript / TypeScript
