# 🎮 Speaking Cards → Phaser 遊戲轉換計劃

## 📋 當前狀態分析

### 現有功能
- ✅ 卡片翻轉機制
- ✅ 語音播放

### 轉換目標

---

## 🎯 Phaser 遊戲架構設計

### 核心場景結構
```
SpeakingCardsScene (extends Phaser.Scene)
├── 初始化 (init)
├── 預加載資源 (preload)
├── 創建遊戲對象 (create)
├── 更新邏輯 (update)
└── 事件處理 (events)
```

### 遊戲對象層次
```
遊戲容器
├── 背景層
├── 卡片堆層
│   ├── 卡片背面 (3 層堆疊)
│   └── 交互區域
├── 發牌區域
│   ├── 卡片正面
│   ├── 圖片
│   ├── 文字
│   └── 語音按鈕
├── UI 層
│   ├── 進度顯示
│   ├── 控制按鈕
│   └── 標題
└── 動畫層
```

## 📁 文件結構

```
public/games/speaking-cards/
├── scenes/
│   ├── game.js          # 主遊戲場景
│   ├── preload.js       # 資源預加載
│   └── ui.js            # UI 管理
├── managers/
│   ├── CardManager.js   # 卡片管理
│   ├── AudioManager.js  # 語音管理
│   └── AnimationManager.js
├── config.js            # 遊戲配置
├── index.html           # 入口 HTML
└── README.md            # 文檔
```
