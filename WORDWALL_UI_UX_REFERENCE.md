## 🖱️ 交互模式

### 1. 詞彙項目操作
```
行操作菜單（右鍵或點擊）：
├── 複製
├── 刪除
├── 上移
├── 下移
└── 編輯詳情

快捷鍵：
├── Ctrl+C: 複製
├── Ctrl+X: 剪切
├── Ctrl+V: 粘貼
├── Delete: 刪除
└── Ctrl+Z: 撤銷
```

### 2. 媒體選擇
```
圖片選擇流程：
1. 點擊圖片單元格
2. 彈出媒體選擇器
   ├── 媒體庫（預設圖片）
   ├── 最近使用
   ├── 上傳新圖片
   └── 搜索

音頻選擇流程：
1. 點擊音頻單元格
2. 彈出媒體選擇器
   ├── 媒體庫（預設音頻）
   ├── 最近使用
   ├── 上傳新音頻
   ├── 錄製新音頻
   └── 搜索
```

### 3. 批量導入
```
導入流程：
1. 點擊「導入」按鈕
2. 選擇文件格式（CSV/Excel）
3. 上傳文件
4. 預覽導入數據
5. 確認導入
6. 顯示導入結果（成功/失敗）
```

## 🎯 用戶流程

### 創建新內容集合
```
1. 點擊「新建內容集合」
2. 輸入標題和描述
3. 選擇語言
4. 創建空集合
5. 開始添加詞彙
```

### 編輯現有內容集合
```
1. 打開內容集合
2. 編輯詞彙項目
3. 添加/刪除項目
4. 上傳媒體
5. 保存更改
```

### 創建活動
```
1. 選擇遊戲模板
2. 選擇內容集合
3. 配置遊戲選項
4. 預覽遊戲
5. 發佈活動
```

## 🎨 設計系統

### 顏色方案
- 主色：#007AFF（藍色）
- 成功：#34C759（綠色）
- 警告：#FF9500（橙色）
- 錯誤：#FF3B30（紅色）
- 背景：#F5F5F5（淺灰）

### 排版
- 標題：18px, 600 weight
- 正文：14px, 400 weight
- 輔助：12px, 400 weight

### 間距
- 小：8px
- 中：16px
- 大：24px
- 超大：32px
