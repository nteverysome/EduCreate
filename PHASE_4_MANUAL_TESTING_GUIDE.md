## 🛠️ 測試工具準備

### 1. 開發服務器

**啟動開發服務器**：
```bash
npm run dev
```

**訪問遊戲**：
```
http://localhost:3000/public/games/match-up-game/index.html
```

### 2. Chrome DevTools

**打開 DevTools**：
- Windows/Linux: F12 或 Ctrl+Shift+I
- Mac: Cmd+Option+I

**關鍵標籤**：
- Console：查看日誌和錯誤
- Elements：檢查 DOM 結構
- Network：檢查資源加載
- Performance：檢查性能

### 3. 設備模擬

**使用 Chrome DevTools 模擬設備**：
1. 打開 DevTools
2. 點擊 Device Toolbar（Ctrl+Shift+M）
3. 選擇設備或自定義解析度
