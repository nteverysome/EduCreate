## 🛠️ 測試工具

### Playwright E2E 測試

**運行方式**：
```bash
# 安裝依賴
npm install -D @playwright/test

# 運行所有測試
npm run test:e2e

# 運行特定測試
npm run test:e2e -- --grep "TC-001"

# 運行並生成報告
npm run test:e2e -- --reporter=html

# 查看報告
npx playwright show-report
```
