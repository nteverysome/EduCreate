# 🎮 统一内容管理系统使用指南

## 🎯 **系统概述**

这个统一内容管理系统实现了你想要的核心功能：
- **一次输入，多游戏复用** - 输入中英文词汇后可以在不同游戏间切换
- **内容持久化存储** - 保存内容，避免重复输入
- **游戏无缝切换** - 内容自动传递到选择的游戏中

## 🚀 **快速开始**

### **第1步：启动系统**
```bash
# 在wordwall-clone目录下运行
node simple-server.js

# 然后访问
http://localhost:3000/unified-content-manager.html
```

### **第2步：输入内容**
1. 填写活动标题和描述
2. 在快速输入区域添加中英文词汇
3. 按Enter键或点击"添加"按钮

### **第3步：保存内容**
- 点击"保存当前内容"按钮
- 内容会保存到浏览器本地存储
- 下次访问时可以直接载入

### **第4步：选择游戏**
- 在游戏选择器中点击想要的游戏类型
- 系统会自动检查词汇数量是否符合要求
- 选择后点击"开始游戏"

### **第5步：享受游戏**
- 内容会自动传递到游戏页面
- 无需重新输入任何内容
- 可以随时返回切换其他游戏

## 🎨 **核心功能详解**

### **📝 内容管理**
```
✅ 快速输入中英文词汇
✅ 实时预览词汇列表
✅ 批量编辑和删除
✅ 自动保存功能（每30秒）
✅ 手动保存和载入
```

### **🎮 游戏支持**
```
❓ 选择题游戏
├── 最少需要：2个词汇
├── 特色功能：计时、积分系统
└── 适合：所有词汇数量

🔗 配对游戏
├── 最少需要：2个词汇
├── 建议最多：16个词汇
├── 特色功能：拖拽操作、视觉反馈
└── 适合：中等数量词汇

🎡 转盘游戏
├── 最少需要：3个词汇
├── 特色功能：物理引擎、音效支援
└── 适合：随机练习

🃏 闪卡游戏
├── 最少需要：1个词汇
├── 特色功能：翻转动画、进度追蹤
└── 适合：记忆训练
```

### **💾 存储系统**
```
📚 已保存内容
├── 显示所有保存的内容集合
├── 点击可快速载入
├── 显示词汇数量和创建时间
└── 支持删除不需要的内容

🔄 自动保存
├── 每30秒自动保存到临时存储
├── 防止意外丢失内容
├── 页面刷新后可恢复
└── 不影响手动保存的内容
```
