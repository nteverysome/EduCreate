## 💡 為什麼選擇 3%？

### 1. 視覺平衡

```
1% 和 2%：太擁擠
4% 和 5%：太寬鬆
3%：完美平衡 ✅
```

### 2. 屏幕利用率

```
1%：可顯示 15 個卡片
2%：可顯示 17 個卡片
3%：可顯示 17 個卡片 ✅
4%：可顯示 18 個卡片
5%：可顯示 19 個卡片

3% 在利用率和視覺效果之間達到平衡
```

### 3. 用戶體驗

```
1%：難以點擊（太擁擠）
2%：有點難點擊
3%：易於點擊 ✅
4%：容易點擊
5%：太容易點擊（浪費空間）

3% 提供最佳的可點擊性
```

### 4. 視覺習慣

```
3% 的間距符合人眼視覺習慣
- 卡片清晰可見
- 間距適中
- 視覺效果舒適
```
