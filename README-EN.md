## 🌟 **Project Overview**

EduCreate is a comprehensive educational game platform focused on creating a Wordwall.net-like SaaS service with unique AI-enhanced features and video processing capabilities. This is an AI-First educational game platform designed to revolutionize online learning experiences.

### 🎯 **Core Features**

- 🤖 **AI Enhancement** - Intelligent content analysis and automatic game generation
- 🎮 **Diverse Games** - Quiz, matching, wheel, flashcards and more game modes
- 📹 **Video Processing** - Advanced video analysis and processing capabilities
- 🌐 **Universal Compatibility** - Multi-platform game content management support
- 📱 **Responsive Design** - Perfect adaptation for desktop and mobile devices
- 🔧 **Modular Architecture** - Easy to extend and customize

## 🚀 **Quick Start**

### **Prerequisites**
- Node.js 18+ 
- PostgreSQL 14+
- Python 3.9+ (for AI features)

### **One-Click Launch**
```bash
# Clone the project
git clone https://github.com/nteverysome/EduCreate.git
cd EduCreate

# Install dependencies
npm install

# Start development server
npm run dev
```

### **Wordwall Clone Quick Experience**
```bash
# Enter wordwall-clone directory
cd wordwall-clone

# Start development server
node server.js

# Visit http://localhost:3000
```

## 🎮 **Core Functionality**

### **1. AI-Enhanced Content Analysis**
- 🔍 **Smart Image Analysis** - Automatically identify and analyze uploaded image content
- 📝 **Auto Vocabulary Extraction** - Intelligently extract learning vocabulary from images
- 🎯 **Game Recommendations** - Automatically recommend the most suitable game modes based on content

### **2. Unified Game Launch System**
- 🌐 **Full Platform Support** - All words automatically adapt to all games
- ⚡ **One-Click Launch** - No manual filtering needed, intelligent content adaptation
- 🎮 **Multiple Game Modes** - Quiz, matching, wheel, flashcards all supported

### **3. Universal Content Management**
- 📚 **Cross-Platform Compatibility** - Support for Wordwall, Kahoot, Quizlet and other platforms
- 🔄 **Data Synchronization** - Unified content format and API interfaces
- 📤 **One-Click Export** - Support for multiple format exports

### **4. Video Processing Capabilities**
- 📹 **Video Analysis** - Intelligent video content analysis
- 🎬 **Auto Subtitles** - AI-generated learning subtitles
- 📊 **Learning Analytics** - Video learning effectiveness analysis

## 🚀 **Getting Started**

1. **Clone the repository**
   ```bash
   git clone https://github.com/nteverysome/EduCreate.git
   ```

2. **Install dependencies**
   ```bash
   cd EduCreate
   npm install
   ```

3. **Set up environment variables**
   ```bash
   cp .env.example .env.local
   # Edit .env.local with your configuration
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```

5. **Visit the application**
   - Main app: http://localhost:3000
   - Wordwall clone: http://localhost:3000/wordwall-clone
