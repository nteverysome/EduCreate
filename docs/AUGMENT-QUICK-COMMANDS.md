# ⚡ Augment 快速命令參考

## 🎯 最常用的 5 個命令

### 1️⃣ 獲取頁面快照
```
請獲取 Responsively App 中遊戲頁面的快照
```
**我會用**: `browser_snapshot_Playwright`

---

### 2️⃣ 獲取控制台日誌
```
請獲取 Responsively App 中的所有控制台日誌，特別是 [v20.0] 和 [v18.0]
```
**我會用**: `browser_console_messages_Playwright`

---

### 3️⃣ 執行 JavaScript 獲取數據
```
請在 Responsively App 中執行 JavaScript，獲取遊戲的視口、卡片、列數信息
```
**我會用**: `browser_evaluate_Playwright`

---

### 4️⃣ 截圖
```
請對 Responsively App 中的遊戲頁面進行截圖，保存為 PNG
```
**我會用**: `browser_take_screenshot_Playwright`

---

### 5️⃣ 點擊操作
```
請在 Responsively App 中點擊第一張遊戲卡片
```
**我會用**: `browser_click_Playwright`

---
