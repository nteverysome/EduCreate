# 📋 項目總覽與目標

## 🎯 項目目標

創建一個功能完整的 Wordwall.net 克隆版本，提供教育工作者創建互動式學習遊戲的平台。

## 📊 目標網站分析

### Wordwall.net 核心特性
- **用戶規模**: 94,004,966+ 個已創建資源
- **模板數量**: 35個互動式模板 + 21個可打印模板
- **核心遊戲**: Quiz, Match up, Flash cards, Spin the wheel, Group sort, Anagram 等
- **用戶功能**: 創建、編輯、分享、嵌入、作業分配
- **多媒體支援**: 文字、圖片、音頻
- **跨平台**: 電腦、平板、手機、互動白板

## 🛠️ 技術棧選擇

### 前端技術
- **框架**: React 18 + TypeScript
- **樣式**: Tailwind CSS + Headless UI
- **動畫**: Framer Motion
- **狀態管理**: Zustand
- **路由**: React Router v6
- **HTTP 客戶端**: Axios + React Query

### 後端技術
- **運行環境**: Node.js 18+
- **框架**: Express.js
- **ORM**: Prisma
- **認證**: JWT + bcrypt
- **實時通信**: Socket.io
- **文件上傳**: Multer + Cloudinary

### 數據庫與存儲
- **主數據庫**: PostgreSQL
- **緩存**: Redis
- **文件存儲**: Cloudinary / AWS S3
- **搜索**: PostgreSQL Full-text Search

### 開發工具
- **版本控制**: Git + GitHub
- **包管理**: npm
- **代碼格式**: Prettier + ESLint
- **測試**: Jest + Cypress
- **API 測試**: Postman

### 部署與監控
- **前端部署**: Vercel
- **後端部署**: Railway / Render
- **數據庫**: Supabase / PlanetScale
- **CDN**: Cloudflare
- **監控**: Sentry + LogRocket
