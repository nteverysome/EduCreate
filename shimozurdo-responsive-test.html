🎯 shimozurdo 響應式核心特性分析

Portrait-First 設計

基準尺寸 540×960 (16:9 Portrait)，專為移動設備優化，確保在手機直向模式下有最佳體驗。

RESIZE Scale Mode

使用 Phaser.Scale.RESIZE 模式，遊戲畫布動態調整到容器尺寸，無黑邊完全填滿。

Math.max 相機算法

camera.setZoom(Math.max(scaleX, scaleY)) 確保遊戲內容完全可見，不會被裁切。

統一 Handler 管理

Handler Scene 集中管理所有場景的響應式行為，確保一致性和可維護性。

雙重 Size 系統

Parent 和 Sizer 分離關注點，使用 Phaser.Structs.Size 進行精確比例計算。

動態 Resize 處理

即時響應窗口大小變化，自動重新計算相機縮放和居中位置。
